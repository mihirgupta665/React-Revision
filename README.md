# React-Revision
Revising one of the important JS UI library.

React is a JS Library for creating UI, developed and maintained by facebook
React is made of  component which could be reused or build on top of one another.
component are piece of UI which have its own login and appearance

JSX : JavScript Extension Syntax
it lets HTML to be written directly inside JS, JSX is transpiled to JS using babel


How a react app start
i> npm create vite@latest, ii> npm install, iii> npm run dev 

How react works(flows)
i> index.html is what reders at the browser and has a script of main.jsx
ii> this script main.jsx renders an App component in the div with root as the id in the index.html
iii>  App is a react component made in the file named app.jsx which gets renedered to div of root id of the index.html file.
iv> styling of app.jsx is mentioned in app.css for the app compenent and that of index.html is mentioned in index.css
